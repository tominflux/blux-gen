# Usage
<!-- usage -->
```sh-session
$ npm install -g blux-gen
$ blux-gen COMMAND
running command...
$ blux-gen (-v|--version|version)
blux-gen/0.1.0 linux-x64 node-v10.13.0
$ blux-gen --help [COMMAND]
USAGE
  $ blux-gen COMMAND
...
```
